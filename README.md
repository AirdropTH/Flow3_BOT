## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/airdropinsiders/LayerEdge-Auto-Bot.git
    cd LayerEdge-Auto-Bot
    ```

2. Install the required dependencies:
    ```sh
    npm install
    ```
3. Fill Data 
    ```sh
    nano token.txt
    ```
4. Run the script:
    ```sh
    node index.js
    ```
